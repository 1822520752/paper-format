## 📖 项目背景

<table>
<tr>
<td width="60%">

### 为什么开发这个工具？

每到毕业季，无数高校毕业生都要面对繁琐的论文格式调整工作：

> 😫 "页边距要调，字体要改，行距要设置……"
> 
> 😫 "参考文献格式又不对，图表标题位置不对……"
> 
> 😫 "改了半天还是不符合学校要求……"

**论文格** 就是为了解决这些痛点而生！

### 我们的目标

- ✅ **解放双手** - 告别繁琐的手动格式调整
- ✅ **提高效率** - 一键完成格式检查与排版
- ✅ **规范统一** - 严格遵循国标 GB/T 7713.1-2006
- ✅ **简单易用** - 无需安装，网页端即可使用

## 🚀 快速开始

### 环境要求

| 环境 | 版本 | 说明 |
|:---:|:---:|:---|
| <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/python/python-original.svg" width="20" alt="Python"> Python | 3.8+ | 推荐 3.10 或 3.11 |
| 📦 pip | 最新版 | Python 包管理器 |
| 💻 操作系统 | 任意 | Windows/macOS/Linux |

### 安装步骤

```bash
# 📥 克隆项目
git clone https://github.com/1822520752/paper-format.git

# 📂 进入后端目录
cd paper-format/backend

# 📦 安装依赖
pip install -r requirements.txt

# 🚀 启动服务
python app.py
```

## 📋 使用指南

<div align="center">

| 步骤 | 操作 | 说明 |
|:---:|:---:|:---|
| **1** | 📤 **上传** | 点击或拖拽 .docx 文件（最大 20MB） |
| **2** | 🔍 **检查** | 点击「格式检查」进行分析 |
| **3** | ✨ **排版** | 点击「一键排版」进行修正 |
| **4** | 📥 **下载** | 获取排版后的规范文档 |

</div>

> ⚠️ **注意**: 仅支持 .docx 格式（Office 2007+），不支持 .doc 格式。

## ⚙️ 配置说明

### 默认配置（国标 GB/T 7713.1-2006）

<div align="center">

| 设置项 | 默认值 | 设置项 | 默认值 |
|:---|:---:|:---|:---:|
| 📏 上页边距 | `2.5 cm` | 📏 下页边距 | `2.5 cm` |
| 📏 左页边距 | `3.0 cm` | 📏 右页边距 | `2.5 cm` |
| 🔤 正文字体 | `宋体 12pt` | 📐 行距 | `1.5倍` |
| 🔤 一级标题 | `黑体 16pt 居中加粗` | 📐 首行缩进 | `2字符` |
| 🔤 二级标题 | `黑体 14pt 左对齐加粗` | 🔤 三级标题 | `黑体 12pt 左对齐加粗` |

</div>

## ❓ 常见问题

<details>
<summary><b>Q：支持哪些文档格式？</b></summary>

<br>

目前仅支持 **.docx** 格式（Office 2007+）。

> ⚠️ .doc 格式（Word 2003 及更早版本）不支持。

如果您的文档是 .doc 格式，请先用 Word 或 WPS 转换为 .docx 格式。

</details>

<details>
<summary><b>Q：会修改我的论文内容吗？</b></summary>

<br>

**不会！** 系统只修改格式（字体、行距、页边距等），不会改动任何文字内容。

您的论文内容 100% 安全。

</details>

<details>
<summary><b>Q：文件安全吗？</b></summary>

<br>

**安全！** 文件仅临时存储，1小时后自动删除。

我们不会保存或备份您的文件。

</details>

<details>
<summary><b>Q：支持哪些标准？</b></summary>

<br>

目前支持：

- **国标 GB/T 7713.1-2006** - 学位论文编写规则
- 自定义高校格式（通过配置实现）

</details>
