使用方法

简单三步，轻松完成论文格式规范化

1

上传论文文档

点击上传区域或拖拽 .docx 格式的论文文档，系统会自动保存并准备检查。支持最大 20MB 的文件。

2

检查格式问题

点击"检查格式"按钮，系统会逐项检查页边距、字体、行距等15+项格式要求，生成详细报告。

3

一键自动排版

点击"一键排版"按钮，系统会自动修正所有格式问题，生成符合规范的新文档供下载。
